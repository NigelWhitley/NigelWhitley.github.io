Two simple programs : each is released under the Gnu GPL.
To see more about them, just click on the associated link.

  • TilePuzzle
    A Java applet which simulates a child's sliding tile puzzle
  • Sudoku helper
    A Javascript aid for solving the logic puzzle known as Sudoku. You can download a zip of the code here
  • Image duplicates
    A PHP program using AJAX which displays duplicate files according to a supplied xml file. Code zip available.

Why have I used the Gnu GPL ?

It protects both me and any users of the code.

It benefits users by granting them the right to use the programs and, if necessary, fix the source code. Of course, that doesn't mean that users are required to do anything with the source code but I compare the GPL to a fire extinguisher: in an ideal world you'll never need it but better to have it and not need it, than need it and not have it.

It benefits me by allowing me to share the source while preserving my rights - it's still copyrighted, not public domain, so my rights have not been diminished. However, for both the Sudoku helper and the TilePuzzle, the most interesting part of developing them was figuring out the internal algorithms. Therefore I want people to be able to see the code so that they can see how they work.
